# 404エラーの解決方法

## 問題

会員登録フォームにアクセスすると「Not Found」エラーが表示される。

## 考えられる原因

### 1. BASE_URLが正しく設定されていない

Renderの環境変数で`BASE_URL`が設定されていない、または間違った値が設定されている可能性があります。

### 2. エンドポイントが正しく登録されていない

`/register`エンドポイントが正しく動作していない可能性があります。

### 3. Renderのルーティングの問題

Renderの設定でルーティングが正しく動作していない可能性があります。

## 確認方法

### ステップ1: Renderのログを確認

1. **Renderのダッシュボードでサービスを開く**
2. **「Logs」タブを確認**
3. **以下のログを探す**:
   - `★フォームアクセス - user_id: ...`
   - `★フォーム表示準備完了 - user_id: ...`
   - エラーメッセージ

### ステップ2: BASE_URLを確認

1. **「Environment」タブを開く**
2. **`BASE_URL`環境変数を確認**
   - 値: `https://line-tool.onrender.com`
   - 正しいURLが設定されているか確認

### ステップ3: ルートパスにアクセス

1. **ブラウザで以下にアクセス**:
   ```
   https://line-tool.onrender.com/
   ```
2. **「LINE Bot 会員登録システム」が表示されるか確認**
3. **表示されない場合は、サーバーが起動していない可能性**

### ステップ4: 直接URLでアクセス

1. **LINE Botから送られてきたURLを確認**
2. **ブラウザで直接アクセス**
3. **エラーメッセージを確認**

## 解決方法

### 方法1: BASE_URLを確認・修正

1. **Renderの「Environment」タブを開く**
2. **`BASE_URL`環境変数を確認**
3. **正しいURLが設定されていない場合、修正**:
   - Key: `BASE_URL`
   - Value: `https://line-tool.onrender.com`
   - （実際のURLに置き換える）

4. **サービスを再起動**
   - 「Manual Deploy」→「Deploy latest commit」

### 方法2: ログを確認

1. **Renderの「Logs」タブを確認**
2. **エラーメッセージの内容を確認**
3. **エラーに応じて対処**

### 方法3: サービスを再起動

1. **「Manual Deploy」→「Deploy latest commit」をクリック**
2. **デプロイが完了するまで待つ**
3. **再度テスト**

## 確認チェックリスト

- [ ] Renderのログでエラーメッセージがないか確認
- [ ] BASE_URL環境変数が正しく設定されているか確認

- [ ] サービスが「Live」状態か確認
- [ ] サービスがスリープしていないか確認
